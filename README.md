# Luxury Motors

Luxury Motors is a premium automotive web platform developed as a full-stack university final project.

The application focuses on delivering a modern luxury dealership experience through dynamic vehicle showcases, immersive UI design and integrated reservation workflows.

## Tech Stack

- Next.js
- React
- MongoDB
- Brevo
- JavaScript
- CSS

## Features

- Dynamic luxury vehicle inventory
- Individual vehicle detail pages
- Purchase and rental workflows
- Reservation and contact forms
- Email integration with Brevo
- Dark mode interface
- Responsive modern UI

## Purpose

This project was designed to transform a traditional academic exercise into a production-oriented full-stack experience focused on premium frontend presentation and real-world integrations.
